---------Image Corpus Creater v0.1.0---------
Welcome to using Kanaan Sullivan's Image Corpus Creator.
When given an input directory, the program will copy the subdirectory format and then adjust and save all images
inside those directories to the specifications given at the beginning of program start with some defaults


-----Usage----
./Corpus (-h) -a -g -r -c -t @indir -outdir
-----Arguments-----
-h -help: Prints the help usage message
-a: Preserve aspect ratio
-g: Save output image as grayscale
-r -rows: Maximum number of rows in the output image (Default: 480)
-c -cols: Maximum number of columns in the output image (Default: 640)
-t -type: Output image type (jpg, png, bmp, tif) (Default: png)
@indir: Input directory containing images
-outdir: Output directory for processed images (default: indir.corpus)
